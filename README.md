# Hola-mundo
JavaScript:
console.log("Hola, mundo");

Python:
print("Hola, mundo")

Java:
public class Main {
public static void main(String[] args) {
System.out.println("Hola, mundo");
}
}

¿Qué hace el programa Hello World?
Sirve como prueba, confirma que el lenguaje está bien instalado, que el código puede ejecutarse sin errores.

¿Qué diferencias visuales notan entre JavaScript, Python y Java?
Que Python y javascrip son mas “minimalistas” solo requieren una sola línea. Y java es el más extenso y visualmente recargado.

¿Qué creen que significa ‘sintaxis’ después de ver estos ejemplos?
Son las reglas gramaticales y de ortografía de un lenguaje de programación. Aunque el objetivo de los tres lenguajes era exactamente el mismo (imprimir un mensaje), cada uno exige palabras clave distintas, un uso diferente de signos de puntuación
